# Primeiro Repositório GitHub

## Descrição
Este é meu primeiro repositório no GitHub, criado com o objetivo de aprender versionamento de código e colaboração em projetos.

## Funcionalidades
- Sistema de Menu interativo
- Módulo de Calculadora
- Funcionalidade de Login
- Gerenciamento de Cadastro

## Como Usar
1. Clone o repositório
2. Execute o programa principal
3. Navegue pelo menu de opções

## Estrutura do Projeto
- menu.txt - Menu principal da aplicação
- cadastro.txt - Módulo de cadastro
- README.md - Documentação

## Como Contribuir
1. Faça um Fork do repositório
2. Crie uma branch para sua feature (git checkout -b feature/minha-feature)
3. Faça commit de suas mudanças (git commit -m 'feat: descrição da mudança')
4. Faça push para a branch (git push origin feature/minha-feature)
5. Abra um Pull Request

## Contribuições
Aberto para sugestões e melhorias! Contribuições são bem-vindas.
